# Prompt — New Tara UI (functionality + keywords)

Paste into Claude (Artifacts / design mode). Describes *what the app does*, not how it looks —
Claude designs the visuals freely.

---

Design a fresh, modern UI for **Tara** — a **Malayalam AI astrology companion** (chat app).
Tone: warm, calm, spiritual, trustworthy — *"guidance, not fear."* Primary language **Malayalam**.

**Keywords:** Kerala · jyothisham · temple · astrology · conversational · calm · warm · trustworthy

**Core functionality to include:**
- **Chat** with Tara: message thread, streaming replies, suggestion chips, follow-up chips,
  per-message actions (copy / retry / thumbs up-down).
- **Sidebar**: new chat, conversation history, "what Tara remembers" (saved user facts), user/logout.
- **AI model selector** in the header.
- **My Chart (jāthakam)** panel: birth details, South-Indian rasi chart, planet table,
  dasha timeline, dosha (present/absent) cards.
- **Panchangam** strip: today's nakshatram / tithi / yogam almanac.
- **Transits & daily reading**: today's star-forecast (nakshatraphalam) grounded in current transits.
- **Temple / vazhipadu suggestions**: recommended temples and offerings for the user's situation.
- **Prashnam** (horary): ask a question via touch-grid / leaf-count / a number 1–108.
- **Porutham** (compatibility): partner birth-details form → 10-porutham match result.
- **Profile**: name, phone, birth details, computed nakshatram / rasi / lagnam / current dasha.
- Persistent **crisis-safety disclaimer** (Tele-MANAS 14416).

**Requirements:** single self-contained file (inline CSS/JS, no external fonts/CDNs), light + dark
themes, responsive, accessible, reduced-motion safe, Malayalam placeholder content, wired mock
interactions. Deliver the artifact + a short rationale.
